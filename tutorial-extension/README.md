# tutorial-extension

A MakeCode Arcade extension used as a starting point for tutorials.

for PXT/TARGET

## Blocks

### ``say hello to $name``

Returns a greeting string for the given name.

```typescript
tutorial.sayHello("Arcade")
// => "Hello, Arcade!"
```

## Usage

Open this repository in the MakeCode Arcade editor (Import from GitHub) to iterate on the extension. See https://makecode.com/extensions/getting-started for the full workflow.
